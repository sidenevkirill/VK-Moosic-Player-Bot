## Описание

VK Moosic Player Console - это консольное приложение для работы с музыкой из ВКонтакте. Программа предоставляет удобный интерфейс для управления вашей музыкальной библиотекой прямо из командной строки.

## ✨ Возможности

- 🎵 **Моя музыка** - доступ ко всем вашим аудиозаписям
- 👥 **Музыка друзей** - прослушивание музыки друзей
- 📋 **Мои плейлисты** - управление плейлистами
- 🔍 **Поиск музыки** - Поиск треков по названию или исполнителю
- 📻 **Рекомендации** - персонализированные рекомендации
